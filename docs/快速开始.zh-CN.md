# 快速开始

## 1. 下载哪些文件

安卓：

- `dist/tp-qr-relay-android-latest.apk`
- `dist/tp-qr-relay-android-latest.apk.sha256`

树莓派固件：

- `dist/system-update-latest.img.xz.part-00`
- `dist/system-update-latest.img.xz.part-01`
- `dist/system-update-latest.img.xz.part-02`
- `dist/system-update-latest.img.xz.sha256`

## 2. Ubuntu 下拼接固件

方法一：

```bash
cd /path/to/tp-satochip-signer/dist
cat system-update-latest.img.xz.part-* > system-update-latest.img.xz
sha256sum -c system-update-latest.img.xz.sha256
```

方法二：

```bash
bash /home/ak/树莓派/tp-satochip-signer/scripts/reassemble_firmware_from_parts.sh
```

成功后会得到：

- `dist/system-update-latest.img.xz`

## 3. 烧录到 TF 卡

先确认 TF 卡设备名：

```bash
lsblk
```

烧录：

```bash
xzcat system-update-latest.img.xz | sudo dd of=/dev/sdX bs=4M status=progress conv=fsync
sync
```

把 `/dev/sdX` 换成你的 TF 卡。

## 4. 安装安卓 App

把 `tp-qr-relay-android-latest.apk` 安装到手机即可。

## 5. 实际使用流程

1. 手机上打开安卓 App。
2. 手机先扫 TP 观察钱包的动态二维码。
3. 手机会自动循环显示给树莓派扫的静态二维码。
4. 树莓派扫完后输入智能卡 PIN。
5. 树莓派签名完成后显示结果二维码。
6. 用 TP 观察钱包直接扫树莓派屏幕上的结果二维码广播。

## 6. 适用范围

已支持：

- EVM 转账
- `personalSign`
- `signTypedData`
- `signTypedDataV4`

## 7. 常见问题

### 树莓派扫不动 TP 动态二维码

这是 Pi Zero 1 性能限制。当前方案就是用安卓 App 先中转，再给 Pi 扫静态码。

### 固件为什么是分片

因为 GitHub 仓库单文件限制 `100MB`，完整固件压缩后仍然超过限制。

### 输入 PIN 后签名失败

先看屏幕上的错误码或错误文字，再按问题定位。当前镜像已经会显示更具体的失败原因。
